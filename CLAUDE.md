# CLAUDE.md

이 파일은 Claude Code (claude.ai/code)가 이 저장소에서 작업할 때 참고하는 안내 문서입니다.

## 프로젝트 개요

유명한 명언을 한국어로 보여주는 단일 페이지 정적 웹사이트입니다. 전체 애플리케이션이 `index.html` 하나에 인라인 CSS와 JavaScript로 구성되어 있으며, 빌드 도구나 외부 의존성, 프레임워크를 사용하지 않습니다.

## 개발 방법

브라우저에서 `index.html`을 직접 열면 됩니다. 별도의 빌드 과정이나 서버가 필요 없습니다.

## 아키텍처

- **index.html**: 인라인 `<style>`과 `<script>` 블록을 포함한 단일 파일 SPA
- 명언 데이터는 JS 배열에 `{ text, author }` 객체로 저장
- 버튼 클릭 시 페이드 애니메이션과 함께 다음 명언으로 전환
- 그라데이션 배경에 글래스모피즘(backdrop-filter) 스타일 적용
